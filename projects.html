Racket Shop

I completed this project as part of my bachelor's degree, with a primary focus on the backend development. However, to further enhance my skills, I took the initiative to incorporate React into the project. Notably, this marked my inaugural experience in seamlessly integrating Laravel and React, showcasing my versatility in both backend and frontend technologies. This application is a Content Management System tailored for administrators, empowering them to efficiently manipulate the database. Additionally, I implemented a compact shop page, providing a user-friendly interface for seamless navigation and interaction.

Appointment App

I successfully crafted this application as a demonstration of my skills during my initial job interview. The primary focus was on showcasing proficiency in creating CRUD applications, utilizing Laravel for the backend and Bootstrap for the frontend. The unique feature of the app was its capability to dynamically filter and display only available dates when customers create new appointments, thereby enhancing user experience.
